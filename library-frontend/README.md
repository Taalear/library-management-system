# 📚 图书管理系统（全栈完整版）

> 一个基于 Spring Boot + Vue 3 的前后端分离图书管理平台，具备完整的权限控制、借还书流程、数据可视化看板和多角色管理能力。

---

## ✨ 项目亮点

- 🔐 **多角色权限控制**：基于 JWT + Spring Security 实现，管理员和普通用户功能隔离
- 📊 **数据可视化驾驶舱**：实时展示图书总数、借阅趋势、热门图书排行榜（ECharts）
- 📖 **完整的借还书闭环**：库存扣减、防超卖、借阅记录追踪、重复借阅校验
- 🎨 **莫兰迪色系 UI**：柔和温馨的视觉风格，配合圆角与阴影，提升用户体验
- 🛡️ **后端专业化改造**：全局异常处理、参数校验（@Valid）、统一日志、自定义业务异常
- 🗂️ **图书分类管理**：图书与分类多对一关联，支持按分类筛选和独立维护分类
- 👤 **用户自助中心**：查看个人信息、修改昵称、修改密码（强制重新登录）
- 📱 **响应式布局**：适配 PC 端和移动端访问

## 📋 核心功能模块

### 1. 认证与权限
- 用户注册 / 登录（JWT 令牌）
- 基于角色的访问控制（RBAC）
    - **管理员（ADMIN）**：数据看板、图书增删改、分类管理、查看所有借阅
    - **普通用户（USER）**：查看图书、借书/还书、我的借阅、个人中心

### 2. 图书管理
- 图书分页查询（支持按书名模糊搜索）
- 按分类筛选图书
- 新增 / 编辑 / 删除图书（仅管理员）
- 库存实时更新

### 3. 借还书流程
- 借书：扣减库存 + 生成借阅记录（事务保证一致性）
- 还书：增加库存 + 更新借阅记录状态
- 防重复借阅（同一本书未归还不能再次借阅）
- 防超卖（SQL 行锁）

### 4. 数据看板（管理员）
- 统计卡片：图书总数、用户总数、总借阅次数、当前借出中
- 近 7 天借阅趋势折线图
- 热门图书 Top 5 柱状图
- 最近借阅动态列表

### 5. 分类管理（管理员）
- 分类的增删改查
- 排序功能（sort_order 控制显示顺序）
- 图书关联分类（新增/编辑时选择）

### 6. 个人中心
- 查看个人信息（用户名、昵称、角色、ID）
- 修改昵称（同步更新侧边栏显示）
- 修改密码（校验原密码 + 强制重新登录）
- 借阅统计（累计借阅、借出中、已归还）
